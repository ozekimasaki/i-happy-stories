## 開発を再開するには

1.  **依存関係のインストール**:
    ターミナルで以下のコマンドを実行し、必要なパッケージがすべてインストールされていることを確認します。
    ```bash
    pnpm install
    ```

2.  **開発サーバーの起動**:
    バックエンドとフロントエンドを同時に起動します。
    ```bash
    pnpm dev
    ```
    ブラウザで `http://localhost:5173` を開くと、現在のアプリケーションが表示されます。
